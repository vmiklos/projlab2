\begin{frame}
\frametitle{Megoldott feladatok}
\framesubtitle{A probléma megoldásához használható algoritmusok tanulmányozása}
Jelenleg elérhetõ, nem VIATRA-n mûködõ algoritmusok feltérképezése:
\begin{itemize}
\item Gráfok összehasonlítására szolgáló, izomorfiát vizsgáló algoritmusok
\item UML modelleket összehasonlító eszközök
\item Fák összehasonlítását megvalósító megoldások
\item Egy-egy megoldás (Odyssey-VCS, diffxml) részletesebb megismerése
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Megoldott feladatok}
\framesubtitle{VIATRA-alapú megoldás elkészítése}
\begin{itemize}
\item A diffxml által használt FMES algoritmus átültetése VIATRA-ra
\item Az algoritmus egy eleme az a döntéshozás, mely eldönti, hogy két elem ugyanaz-e különbözõ modellekben
\item Itt figyelembe lehetne venni az XML sémát, jelenleg ez nincs így
\item Az algoritmus a modellben beszúrásokat, áthelyezéseket, törléseket detektál
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Megoldott feladatok}
\framesubtitle{XML importer}
\begin{itemize}
\item Az FMES teszteléséhez szükség volt teszt modellekre
\item A diffxml projekt már rendelkezett egy ilyen tesztgyûjteménnyel
\item A gyûjtemény elemei XML fájlok, ehhez kell egy VIATRA importer
\item Importer séma nélküli XML-ekhez
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Tesztelési környezet}
\begin{itemize}
\item Eclipse 3.5
\item VIATRA2 SVN (r1334)
\item diffxml 0.95B
\item Kézi tesztelés (runtime eclipse, importálás, model comparator futtatása)
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Jövõbeli lehetõségek}
\begin{itemize}
\item XML importer: Séma megkövetelése, így a VIATRA-beli entitásokhoz típusok lennének rendelhetõek
\item Csak a comparator került megvalósításra, ennek kimenete alapján egy modell egyesítõ eszközt is lehetne készíteni
\item A két entitást összehasonlító elem finomítható lenne
\item Az FMES fákon dolgozik, egyéb gráf-összehasonlító algoritmusok is elérhetõek (pl. Groove-ból)
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Összefoglalás}
\begin{itemize}
\item A modellek összehasonlítása és egyesítése a modellvezérelt fejlesztés ma egyik aktuális kérdése
\item Részmegoldások elérhetõek: Odyssey-VCS, diffxml, Groove, stb.
\item A labor keretében elkészült egy általános XML importer és egy FMES-alapú modell összehasonlító a VIATRA rendszerhez.
\item Számos jövõbeli lehetõség mert fel a félév során, melyek még megvalósításra várnak
\end{itemize}
