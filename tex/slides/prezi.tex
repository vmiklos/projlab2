\begin{itemize}
\item A dm-mirror egy tipikus 3 rétegû architektúra legalsó szintje \emph{alatt} helyezkedik el
\item Egy szoftveres tükrözési megoldás a Linux kernelben
\item Egyedülálló tulajdonsága, hogy a device-mapper keretrendszerre épül (logikai kötetkezelés)
\item Kezdeti probléma: szuboptimális teljesítmény ha a fizikai diszkek elérési sebessége nem azonos
\end{itemize}
\end{frame}

\begin{itemize}

\item Linux kernel része: szabadon hozzáférhetõ és módosítható
\item Az elkészült munkát hoszútávon cél visszajuttatni a kernelbe
\end{itemize}
